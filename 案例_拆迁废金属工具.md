# 🏗️ 拆迁废金属综合工具 - 项目案例

## 项目背景

**客户需求：** 做拆迁拆除工程的公司，需要：
1. 第一时间获取拆迁招标信息
2. 产出废钢/废铜/废铝后能及时了解价格
3. 形成生意闭环

## 解决方案

```
🔍 查招标 (TenderAI) → 🏆 中标做拆除 → 🔩 产出废料 → 💰 富宝查价卖出
```

## 技术架构

```
┌─────────────────────────────────────────────────────────┐
```

## 功能演示

### 1. 招标查询
- 自动抓取政府采购网招标信息
- 关键词：拆迁、拆除、旧城改造、棚户区改造...
- 地区：温州市、浙江省、全国

### 2. 金属价格监控
- 废钢、废铜、废铝、废不锈钢
- 实时价格 + 涨跌情况
- 数据来源：富宝资讯

### 3. 微信推送通知
- 每天自动发送报告到微信
- 包含招标信息 + 价格数据
